Cardify

About & Info

This project was based around trainer cards from the Generation 3 Pokemon Games (namely Pokemon Emerald).


This website does not collect or save any data from your spotify account, all data is hosted locally on the client side.

All of the code that this website runs on is avalible on Github.
Popularity is calculated as an average of 50 top tracks of the last 6 months for a user.
